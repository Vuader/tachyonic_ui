Tachyonic UI User Guide
=======================

Welcome to the Tachyonic User Guide. These pages lists general usage guidelines on the Tachyonic Web User Interface.

Introduction
------------
When visiting the URL of the Tachyonic installation in a web browser, one is presented with a login screen.
The default username is *root*, and the default password is *password*.

The third field can be used to log into a specific :ref:`domain <domains>`, and is optional. If left blank, one will log into the *default* domain.
